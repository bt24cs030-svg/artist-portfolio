# 🎨 ArtHub Backend

A robust backend API for an art marketplace platform where artists can showcase and sell their artworks.

---

## 🚀 Tech Stack
- Node.js
- Express.js
- MySQL
- JWT Authentication
- bcrypt (password hashing)

---

## ✨ Features
- 👤 Artist Registration & Login
- 🔐 Secure Authentication using JWT
- 🎨 Artwork Upload & Management
- 🛒 Order Management System
- ⭐ Reviews & Ratings System
- 📊 Artist Dashboard
- 🔍 Search & Filter Artworks
- ⚡ Pagination Support
- 🛡️ Security (Helmet, Rate Limiting)

---

## 📂 API Endpoints (Sample)
- `POST /api/register` → Register artist
- `POST /api/login` → Login
- `GET /api/artists` → Get all artists
- `POST /api/artworks` → Upload artwork
- `POST /api/orders` → Create order
